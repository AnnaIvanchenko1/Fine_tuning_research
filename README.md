# Исследование различных подходов к обучению моделей классификации изображений на датасете Fantastic-Beasts

# Стек
Python, PyTorch, Hugging Face Transformers, ViT (Vision Transformer), LoRA, Transfer Learning, scikit-learn, pandas, NumPy

# Описание проекта
Данный проект посвящен реализации и сравнительному анализу нескольких методов обучения моделей для задачи классификации изображений на базе датасета Fantastic-Beasts. В рамках проекта используются различные подходы, включая:

- Обучение без дообучения (vanilla training) на предварительно замороженной модели ViT.
- Fine-tuning модели с использованием методов LoRA (Low-Rank Adaptation) для адаптации модели к специфике датасета.
- Transfer Learning с предварительной настройкой и последующим обучением на целевом датасете.
- Incremental Learning — постепенное обновление модели по мере появления новых данных или категорий без необходимости полного переобучения с нуля.

Проект включает подготовку датасета, обучение моделей, их оценку и сравнение по метрикам точности.

# Цель проекта
Цель проекта — создать эффективные модели для классификации изображений, способные точно распознавать категории по датасету Fantastic-Beasts, и проанализировать влияние различных методов обучения на качество модели. В частности, планируется:

- Исследовать влияние обучения без дообучения и с дообучением с помощью LoRA.
- Оценить эффективность transfer learning по сравнению с полностью обученными моделями.
- Обеспечить возможность быстрого прототипирования и разработки моделей с учетом ограниченных ресурсов.
- Внедрить методы инкрементального обучения для динамического обновления модели по мере поступления новых данных или категорий без необходимости полного переобучения.

# Основные компоненты системы

- Датасет: изображения персонажей и объектов из вселенной Fantastic-Beasts, разбитые на обучающую и тестовую выборки.

- Модель: предварительно обученная модель ViT (Vision Transformer) из Hugging Face.

- Обучение:
   - Обучение без дообучения (замороженная модель).
   - Fine-tuning с использованием LoRA для адаптации модели к новому датасету.
   - Transfer learning — обучение всей модели без заморозки слоёв.
   - Incremental Learning - постепенное добавление новых данных или категорий с обновлением модели без полного переобучения.
     
- Оценка: метрики точности, loss, confusion matrix.

# Используемые данные

- Датасет изображений из серии Fantastic-Beasts, разбитый на train и test множества.
- Метки категорий для обучения и оценки модели.

# Результаты и выводы

- Обученные модели показывают различную точность в зависимости от выбранного метода.
- Использование LoRA значительно ускоряет адаптацию модели при сохранении высокого качества.
- Transfer Learning позволяет добиться хорошей производительности при меньших затратах времени и ресурсов.
- Incremental Learning даёт возможность обновлять модель по мере появления новых данных или категорий, что особенно важно для динамичных или расширяемых систем.

В итоге получены модели, пригодные для автоматической классификации изображений персонажей и объектов из вселенной Fantastic-Beasts.
